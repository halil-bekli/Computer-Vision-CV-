# Computer Vision (CV)

Projects for learning and applying modern Computer Vision techniques. This repository serves as a practical guide to understanding and implementing various CV concepts in real life.

**Tech Stack:**

*   **Language:** Python
*   **Platform:** Anaconda
*   **Image Processing:** OpenCV
*   **Matrix Operations:** NumPy
*   **Visualization:** Matplotlib
*   **IDE:** Spyder

- **OpenCV Object Detection** 
- **OpenCV Object Tracking** 
- **Convolutional Neural Networks (CNN)** 
- **Object Detection with CNN** 
- **R-CNN Architecture** 
- **YOLO (You Only Look Once) Object Detection** 
- **SSD (Single Shot MultiBox Detector) Object Detection**
